# 6.104 Forkcast Backend

## Concepts index

Each concept lives under `src/concepts/<ConceptName>/`. Common files in a concept folder:

- `<ConceptName>Specification.md` — specification (markdown)
- `<ConceptName>Concept.ts` — implementation (TypeScript)
- `<ConceptName>Concept.test.ts` — tests (TypeScript/Deno)
- `<ConceptName>ConsoleOutput.md` — recorded test console output (markdown)
- `<ConceptName>Changes.md` — design notes and changes (markdown)
